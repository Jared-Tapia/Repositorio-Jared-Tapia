Tema 1: Objetos

¿Qué es un objeto en JS?

Los objetos de Java Script, son contenedores para valores con nombre llamados propiedades.

Diferencia entre objeto y variable:

Las variables de Java Script son contenedores para valores de datos. Los objetos también son variables; pero los objetos pueden contener muchos valores.

¿Cómo y con qué se crea un objeto en JS?

Un objeto de JS se crea con objeto literal.

const person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};

Ejemplo de cómo crear un objeto en JS:

const persona = {firstName:"Jared", lastName:"Tapia", age:16, eyeColor:"brown"};

¿Para que se utiliza document.getElementById ("demo").innerHTML?

Es un metodo que permite seleccionar un elemento del documento por medio del valor del atributo asignado.
